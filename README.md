# python_weather
## Requirements
-   Python 3
-   PIP 3
-  Virtual Enviroments

## Instructions

  -   Clone the repository
``git@github.com:vcjuliocesar/python_weather.git``
 - Create Virtual Enviroment
  ```python3  -m  menv  env```
  
- Activate virtual enviroment
``source  env/bin/activate``
- Install dependencies
``pip3  install  -r  requeriments.txt``
- Run project
``python3  main.py``
